A plain language description of the FHC Committee model

Document

Ο 1. Each Document may have a Name, which carries a String value.
Ο 2. Each Document may have a Description, which carries a String value.
Ο 3. Each Document may have a Doc File, which carries a String value.
Ο 4. Each Document may have a Seq No, which carries a int value.

Event

Ο 5. Each Event may have a Year, which carries a int value.
Ο 6. Each Event may have an Objectives, which carries a String value.

Ο 7. Each Event can have zero or more assignments (RoleAssignment).

Announcement

Ο 8. Each Announcement may have a Topic, which carries a String value.
Ο 9. Each Announcement may have a Description, which carries a String value.
Ο 10. Each Announcement may have a Seq No, which carries a int value.

Volunteer

Ο 11. Each Volunteer may have a L Name, which carries a String value.
Ο 12. Each Volunteer may have a F Name, which carries a String value.
Ο 13. Each Volunteer may have a Full Name, which carries a String value.
Ο 14. Each Volunteer may have an Email, which carries a String value.
Ο 15. Each Volunteer may have a Email Confirmed, which carries a boolean value.
Ο 16. Each Volunteer may have a User Name, which carries a String value.
Ο 17. Each Volunteer may have a Passwd, which carries a String value.
Ο 18. Each Volunteer may have a Address Line1, which carries a String value.
Ο 19. Each Volunteer may have a Address Line2, which carries a String value.
Ο 20. Each Volunteer may have a City, which carries a String value.
Ο 21. Each Volunteer may have a State, which carries a String value.
Ο 22. Each Volunteer may have a Zipcode, which carries a String value.
Ο 23. Each Volunteer may have a Cell Phone, which carries a String value.
Ο 24. Each Volunteer may have a Home Phone, which carries a String value.
Ο 25. Each Volunteer may have a Work Phone, which carries a String value.
Ο 26. Each Volunteer may have a Missionary Ind, which carries a boolean value.
Ο 27. Each Volunteer may have a Created At, which carries a Date value.

Ο 28. Each Volunteer can have zero or more assignments (RoleAssignment).

Meeting

Ο 29. Each Meeting may have a Name, which carries a String value.
Ο 30. Each Meeting may have a Purpose, which carries a String value.
Ο 31. Each Meeting may have an Agenda, which carries a String value.
Ο 32. Each Meeting may have a Month Week, which carries a int value.
Ο 33. Each Meeting may have a Notes, which carries a String value.
Ο 34. Each Meeting may have a Seq No, which carries a int value.

Ο 35. Meetings may have zero or more invitees (CommitteeRole). Also, each CommitteeRole can have zero or more meetings (Meetings).

Ο 36. Meetings may have zero or more resources (Resource). Also, each Resource can have zero or more meeting (Meetings).

Ο 37. Each Meeting can have zero or more tasks_addressed (TaskObj).

Task

Ο 38. Each Task may have a Name, which carries a String value.
Ο 39. Each Task may have a Description, which carries a String value.
Ο 40. Each Task may have a Month Week, which carries a int value.
Ο 41. Each Task may have a Key Task, which carries a boolean value.
Ο 42. Each Task may have a Seq No, which carries a int value.

Ο 43. Tasks may have zero or more documents (Document). Also, each Document can have zero or more tasks (TaskObjs).

Ο 44. Each Task can have zero or more announcements (Announcement).

Ο 45. Each Task can have zero or more follow_on_tasks (TaskObj).

Ο 46. Tasks may have zero or more resources (Resource). Also, each Resource can have zero or more tasks (TaskObjs).

Ο 47. Each Task can have zero or more functions (WebsiteFunction).

Committee

Ο 48. Each Committee may have a Code, which carries a String value.
Ο 49. Each Committee may have a Name, which carries a String value.
Ο 50. Each Committee may have a Description, which carries a String value.
Ο 51. Each Committee may have a Picture, which carries a String value.
Ο 52. Each Committee may have a Seq No, which carries a int value.

Ο 53. Each Committee can have zero or more tasks (TaskObj).

Ο 54. Each Committee can have zero or more committee_roles (CommitteeRole).

Website Function

Ο 55. Each Website Function may have a Name, which carries a String value.
Ο 56. Each Website Function may have a Description, which carries a String value.
Ο 57. Each Website Function may have a Link, which carries a String value.
Ο 58. Each Website Function may have a Seq No, which carries a int value.

Committee Role

Ο 59. Each Committee Role may have a Role Name, which carries a String value.
Ο 60. Each Committee Role may have a Description, which carries a String value.
Ο 61. Each Committee Role may have a Calling Required, which carries a boolean value.
Ο 62. Each Committee Role may have a Name Tag Needed, which carries a boolean value.
Ο 63. Each Committee Role may have a Seq No, which carries a int value.

Ο 64. Each Committee Role can have zero or more meetings (Meeting).

Ο 65. Each Committee Role can have zero or more tasks (TaskObj).

Ο 66. Each Committee Role can have zero or more assignments (RoleAssignment).

Role Assignment

Ο 67. Each Role Assignment may have a Name, which carries a String value.

Ο 68. Each Role Assignment can have zero or more assigned_committees (Committee).

Resource

Ο 69. Each Resource may have a Name, which carries a String value.
Ο 70. Each Resource may have a Description, which carries a String value.
Ο 71. Each Resource may have a Picture, which carries a String value.
Ο 72. Each Resource may have a Doc File, which carries a String value.
Ο 73. Each Resource may have a Link, which carries a String value.
Ο 74. Each Resource may have a Seq No, which carries a int value.
